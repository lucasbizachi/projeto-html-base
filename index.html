Estrutura básica.

Voltar

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto, em português). É uma linguagem de marcação utilizada na construção e estruturação de páginas web. Desenvolvido para descrever a estrutura básica de uma página web, o HTML consiste em uma série de elementos ou tags que são utilizados para definir diferentes partes do conteúdo, como cabeçalhos, parágrafos, listas, links, imagens, formulários, entre outros.

As páginas web são compostas por texto, imagens, links e outros elementos, e o HTML fornece a estrutura necessária para organizar e apresentar esses elementos de forma coerente. Além do HTML, o desenvolvimento web geralmente envolve o uso de outras tecnologias, como o CSS (Cascading Style Sheets) para a estilização e o layout, e o JavaScript para a interatividade e dinamismo na página.

Em resumo, o HTML é uma linguagem fundamental para a criação de páginas web, pois fornece a base estrutural necessária para a apresentação de conteúdo na internet.

Tags

As tags em HTML são elementos fundamentais dessa linguagem de marcação. Elas são usadas para marcar partes específicas do conteúdo de uma página web, indicando ao navegador como essas partes devem ser exibidas ou como devem interagir entre si. Cada tag HTML é representada por uma palavra-chave cercada por colchetes angulares < >. Geralmente, as tags HTML são usadas em pares, com uma tag de abertura e uma tag de fechamento, que são idênticas, exceto pela barra inclinada (/) antes do nome da tag de fechamento.

Algumas tags, como a <img> para imagens ou a <br> para quebras de linha, não precisam de uma tag de fechamento e são chamadas de tags vazias.

Exemplo:

<p>Este é um parágrafo.</p>
<img src="" alt="">

Em HTML, os atributos são utilizados para fornecer informações adicionais sobre elementos HTML. Eles geralmente são incluídos nas tags de abertura dos elementos e fornecem configurações ou características específicas para esses elementos. Aqui estão alguns exemplos de atributos HTML comuns:

Estes são apenas alguns exemplos, e existem muitos outros atributos HTML que são específicos para diferentes elementos. A utilização adequada de atributos é essencial para estruturar e estilizar corretamente o conteúdo em uma página web.

Atributo src (source):

Utilizado em elementos como <img>, <audio>, e <video>. Define o caminho ou URL da origem do conteúdo.
Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">

Atributo alt (alternative text):

Utilizado em elementos como <img> para fornecer um texto alternativo quando a imagem não pode ser exibida.
Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">

Atributo href (hypertext reference):

Usado em links para especificar o destino do link.
Exemplo: <a href="https://www.exemplo.com">Visite nosso site </a >

Atributo class:

Permite associar um ou mais elementos a uma classe, que pode ser estilizada através de CSS.
Exemplo: <div class="destaque">Este é um bloco com destaque.</div>

Atributo id:

Atribui uma identificação única a um elemento, o que pode ser útil para referenciá-lo com estilos CSS ou scripts.
Exemplo:<div id="conteudo-principal">Conteúdo principal da página.</div>

Em HTML, as listas são usadas para organizar e estruturar informações de maneira hierárquica. Existem três tipos principais de listas em HTML: listas ordenadas, listas não ordenadas e listas de definição.

Listas Ordenadas (<ol> - Ordered List):

Utilizadas para criar listas com itens numerados ou ordenados.

Utilizadas para criar listas com itens sem uma ordem específica, geralmente marcados com pontos, traços ou outros símbolos.

Resultado:

  • Item A
  • Item B
  • Item C

Listas de Definição (<dl> - Definition List):

Utilizadas para criar listas de termos e suas definições associadas. Cada definição é representada por um par de tags <dt> (termo) e <dd>(definição).
